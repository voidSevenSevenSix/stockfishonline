A bit about the project

Hello, fellow developers and Chess players! You have probably come here in the search of a simple way to add Stockfish to your application, because this is way harder than it should be.

When I made this project, I needed an API like this but simply couldn't find one. The closest thing there is can be found at https://www.chessdb.cn/cdb.php, which isn't actually Stockfish, but rather a database of positions. However, if your position isn't in the database, it can't help you.

Quick shoutout to hyugit on Github, who wrote the nodejs program which creates a Stockfish server locally. The code did require tweaking, but was functional enough that I could make do and expand upon it.

Then I wrote up a PHP program to query and serve data from the locally hosted Stockfish, and it was done!

Lastly, please excuse my CSS skills :) Not something I ever put any effort into as I'm more of a backend guy...
